Problem: Doanh nghiệp cần môi trường chạy mô hình ngôn ngữ lớn (LLMs) nội bộ để đảm bảo bảo mật dữ liệu tuyệt đối. Cần tận dụng tối đa card đồ họa rời (NVIDIA GTX 1060) trên máy chủ vật lý đang chạy nền tảng ảo hóa.

DevOps Solution:
1. Cấu hình IOMMU trên Proxmox và áp dụng kỹ thuật PCIe Passthrough (Raw Device) để đẩy thẳng GPU vật lý vào Ubuntu VM.
2. Troubleshooting sâu ở tầng OS: Khắc phục sự cố xác thực SSH do Cloud-Init cache (override file sshd_config.d) và fix lỗi Error 43 của NVIDIA trên môi trường ảo hóa bằng cờ CPU Host.
3. Triển khai CUDA Toolkit, Ollama và model Llama-3. Kết quả: Nạp thành công model 4.7GB vào VRAM, đẩy 100% tải suy luận (Inference) sang GPU giúp tốc độ phản hồi cực kỳ nhanh chóng.
